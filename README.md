# Weather Dashboard

## Description
This is a simple and interactive Weather Dashboard application built using HTML, CSS, and JavaScript. It allows users to view current weather conditions, a 5-day forecast, and an hourly forecast for various cities in Pakistan. The dashboard features a dynamic background that changes based on the weather conditions and time of day, and also allows toggling between Celsius and Fahrenheit temperatures.

## Features
- **Current Weather Display**: Shows current temperature, weather description, high/low temperatures, humidity, wind speed, pressure, and UV index.
- **City Selection**: Users can select from a predefined list of cities in Pakistan (Lahore, Karachi, Islamabad, Peshawar, Quetta, Gilgit, Muzaffarabad).
- **Dynamic Backgrounds**: The background image of the dashboard changes based on the current weather condition (sunny, cloudy, rainy, thunderstorm, windy) and time of day (day/night).
- **Temperature Toggle**: Switch between Celsius and Fahrenheit temperature units.
- **5-Day Forecast**: Provides a daily forecast including high/low temperatures and weather conditions for the next five days.
- **Hourly Forecast**: Displays hourly temperature and weather conditions for the next 24 hours.
- **Responsive Design**: The dashboard is designed to be visually appealing and functional across different screen sizes.

## Technologies Used
- **HTML5**: For structuring the web content.
- **CSS3**: For styling the application, including responsive design and dynamic backgrounds.
- **JavaScript**: For handling user interactions, fetching and displaying weather data (mock data used), and dynamically updating the UI.

## Usage
- **Select a City**: Use the dropdown menu in the header to choose a city and view its weather information.
- **Toggle Temperature Unit**: Click the 'C' or 'F' button next to the main temperature display to switch between Celsius and Fahrenheit.
- **View Forecasts**: Scroll down to see the 5-day daily forecast and the 24-hour hourly forecast.
